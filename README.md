# 🍽️ Restaurant Operations System

A comprehensive **Data Engineering & Business Intelligence** project that demonstrates the complete data analytics pipeline—from data preparation and ETL workflows to interactive business dashboards.

This project was developed using **Power BI**, **KNIME**, **Excel**, and **SQL** to analyze restaurant operations, financial performance, order management, banking reconciliation, and business insights.

---

## 📌 Project Overview

The Restaurant Operations System (ROS) is designed to help restaurant businesses monitor daily operations through interactive dashboards and automated workflows.

The project includes:

- Data cleaning and validation
- ETL workflow development using KNIME
- Data modeling and transformation
- Business Intelligence dashboard using Power BI
- KPI monitoring
- Financial analysis
- Banking reconciliation
- Interactive reporting

# 🔄 KNIME Workflows

The project includes five ETL workflows:

1. Customer Places an Order
2. Delivery Partner Delivers Order
3. Cash-Up Publication
4. Banking Process
5. Client Subscription Management

These workflows automate data validation, business rules, financial calculations, and operational processing.
